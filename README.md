# generator-mean2
Yeoman generator to generate a MEAN stack application for Angular 2 and TypeScript

# about
MEAN stack development with angular2

Includes:
- Angular 2
- TypeScript
- MongoDB
- Express
- SASS
- Gulp

# getting started

Install Yeoman
```
$ npm install -g yo
```

Install the generator
```
$ npm install -g generator-mean2
```

Run the generator
```
$ yo mean2
```
